#### 1.模板字符串

模板字符串``

##### @1.可以很轻松的表达字符串中的变量

以前拼接字符串和变量需要用很多引号和加号,现在只需要给变量套一个

```
`$(变量) 壳子即可`
```

##### @2.反引号需要转义

```
因为模板字符串是由反引号组成,如果内容包含`, 需要再前面添加转义符号
\`
```
