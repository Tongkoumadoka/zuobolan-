#### 1.计算机常用快捷键

```
1.alt + F4
关闭整个窗口(即使有多个标签页)
```

---

```
2.文件永久删除
选中文件 + 长按shift键 + delete

作用:会直接删除文件,而不放到回收站
```

---

```
3.运行
win + R

作用:跳出运行窗口(cmd)
```

```
4.打开我的电脑
win + E
```

---

```
5.返回桌面
win + D
```

---

```
6.任务管理器
Ctrl + shift +esc
```
